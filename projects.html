Discord Voting Bot

This is a Discord bot that allows users to submit and vote on anything. All basic functionality works, but it still needs to take into account administration privileges.

Because it was programmed with Python, the bot can react slowly to commands. Sometimes it can take upto 3 seconds to respond, which is okay if only one user is using it... but is not desirable when multiple users are voting at the same time.

The solution would appear to be to use a faster language, like JS or C++. Because of the speed issue, it is unlikely I will update this bot any further.

Mastermind Game

This project is based on the board game "Mastermind". Two players choose a secret code and attempt to crack the code of their opponent by using hints that come in the form of white or black pegs.

The code must be guessed within 10 tries or the player will no longer be able to guess. Whichever player guesses the code in the least amount of turns first wins.

This game was created in Python, and does not have a GUI.
